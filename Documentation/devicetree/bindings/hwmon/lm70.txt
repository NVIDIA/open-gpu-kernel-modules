* LM70/TMP121/LM71/LM74 thermometer.

Required properties:
- compatible: one of
		"ti,lm70"
		"ti,tmp121"
		"ti,tmp122"
		"ti,lm71"
		"ti,lm74"

See Documentation/devicetree/bindings/spi/spi-bus.txt for more required and
optional properties.

Example:

spi_master {
	temperature-sensor@0 {
		compatible = "ti,lm70";
		reg = <0>;
		spi-max-frequency = <1000000>;
	};
};
