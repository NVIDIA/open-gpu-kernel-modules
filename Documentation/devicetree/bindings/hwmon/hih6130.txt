Honeywell Humidicon HIH-6130 humidity/temperature sensor
--------------------------------------------------------

Requires node properties:
- compatible : "honeywell,hi6130"
- reg : the I2C address of the device. This is 0x27.

Example:
	hih6130@27 {
		compatible = "honeywell,hih6130";
		reg = <0x27>;
	};
