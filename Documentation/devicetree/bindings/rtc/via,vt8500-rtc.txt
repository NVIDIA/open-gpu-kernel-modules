VIA/Wondermedia VT8500 Realtime Clock Controller
-----------------------------------------------------

Required properties:
- compatible : "via,vt8500-rtc"
- reg : Should contain 1 register ranges(address and length)
- interrupts : alarm interrupt

Example:

	rtc@d8100000 {
		compatible = "via,vt8500-rtc";
		reg = <0xd8100000 0x10000>;
		interrupts = <48>;
	};
