Qualcomm PM8xxx PMIC Vibrator

PROPERTIES

- compatible:
	Usage: required
	Value type: <string>
	Definition: must be one of:
		    "qcom,pm8058-vib"
		    "qcom,pm8916-vib"
		    "qcom,pm8921-vib"

- reg:
	Usage: required
	Value type: <prop-encoded-array>
	Definition: address of vibration control register

EXAMPLE

	vibrator@4a {
		compatible = "qcom,pm8058-vib";
		reg = <0x4a>;
	};
