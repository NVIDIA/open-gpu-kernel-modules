* Microchip AR1020 and AR1021 touchscreen interface (I2C)

Required properties:
- compatible		: "microchip,ar1021-i2c"
- reg			: I2C slave address
- interrupts		: touch controller interrupt

Example:

	touchscreen@4d {
		compatible = "microchip,ar1021-i2c";
		reg = <0x4d>;
		interrupt-parent = <&gpio3>;
		interrupts = <11 IRQ_TYPE_LEVEL_HIGH>;
	};
