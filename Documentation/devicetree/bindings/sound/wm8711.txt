WM8711 audio CODEC

This device supports both I2C and SPI (configured with pin strapping
on the board).

Required properties:

  - compatible : "wlf,wm8711"

  - reg : the I2C address of the device for I2C, the chip select
          number for SPI.

Example:

wm8711: codec@1a {
	compatible = "wlf,wm8711";
	reg = <0x1a>;
};
