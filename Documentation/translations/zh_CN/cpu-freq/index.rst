=======================================================
Linux CPUFreq - Linux(TM)内核中的CPU频率和电压升降代码
=======================================================

Author: Dominik Brodowski  <linux@brodo.de>

      时钟升降允许你在运行中改变CPU的时钟速度。这是一个很好的节省电池电量的方法，因为时
      钟速度越低，CPU消耗的电量越少。
